# ✈️ Análise de Atrasos Aéreos (Logística)

Este projeto apresenta uma análise detalhada dos atrasos em voos, transformando dados brutos em insights estratégicos para a gestão de malha aérea.

## 🛠️ Tecnologias Utilizadas
* **SQL (PostgreSQL):** Limpeza de dados, tratamento de valores nulos e criação de regras de negócio.
* **Power BI & Power Query:** ETL (Extração, Transformação e Carga) e modelagem de dados.
* **DAX:** Criação de medidas de performance (KPIs) como Média de Atraso.

## 📈 Insights do Dashboard
* Identificação das companhias aéreas com maior índice de atraso crítico.
* Análise temporal de atrasos por mês utilizando segmentação de dados.
* Visualização da média de atraso geral (atualmente em 422 minutos no dataset analisado).

## 📂 Estrutura do Repositório
* `/02_Scripts_SQL`: Contém o script de tratamento de dados.
* `/03_Dashboard_PBI`: Arquivo .pbix com o relatório final.
